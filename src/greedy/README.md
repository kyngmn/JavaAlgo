## 그리디

최적의 선택을 보장하는 방식
불필요한 연산 없이 최소한의 연산 횟수를 구할 수 있다.

- **항상** 작은 값에 큰 값을 더하는 방식
- 작은 값, 큰 값 유지하도록 **정렬**

### 언제 쓸까?

1. 현재 최선의 선택만으로 최적해를 보장하는 경우

2. 부분 문제 최적해가 전체 문제 최적해로 이어지는 경우

3. 한번 선택한 결과를 되돌릴 필요 없는 경우

### 대표 문제

1. 거스름돈 문제

2. 회의실 배정 문제

3. 최소 신장 트리 (Kruskal, Prim 알고리즘)

4. Dijkstra's Algorithm

5. 배낭 문제
